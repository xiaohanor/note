## 工程项目 \[#工程项目]

1. 虚幻引擎官方 ARPG 项目社区维护版，支持最新版本 UE5，适合入门学习 GAS： [vahabahmadvand/ActionRPG_UE5: Action RPG sample project upgraded to the latest Unreal Engine, It works with UE 5.3, 5.4 and 5.5. It is a great leaning resource for GAS](https://github.com/vahabahmadvand/ActionRPG_UE5)
2.  高级运动系统(ALS) 的 C++ 修复版，支持 UE5 和增强输入系统：[ShadowfallStudios/ALS-Community: Replicated and optimized community version of Advanced Locomotion System V4 for Unreal Engine 5.4 with additional features & bug fixes](https://github.com/ShadowfallStudios/ALS-Community)
3. 更轻量的空白项目：[daftsoftware/StarterProject: A lightweight uproject that debloats Unreal Engine, reducing package size and increasing editor and game performance.](https://github.com/daftsoftware/StarterProject)
