---
title: Union
---
联合（union）是一种节省空间的特殊的类，一个 union 可以有多个数据成员，但是在任意时刻只有一个数据成员可以有值。当某个成员被赋值后其他成员变为未定义状态。联合有如下特点：

* 默认访问控制符为 public
* 可以含有构造函数、析构函数
* 不能含有引用类型的成员
* 不能继承自其他类，不能作为基类
* 不能含有虚函数
* 匿名 union 在定义所在作用域可直接访问 union 成员
* 匿名 union 不能包含 protected 成员或 private 成员
* 全局匿名联合必须是静态（static）的

***

在C++中，union和struct是两种不同的数据结构，它们有一些重要的区别。

1. 内存分配方式：
   * Union：所有成员共享同一块内存。这意味着如果你改变一个成员的值，其他成员的值也会发生变化。
   * Struct：每个成员都有自己独立的内存空间，它们不共享内存。

2. 内存占用：
   * Union：占用的内存空间等于最大成员的内存空间。
   * Struct：占用的内存空间等于所有成员的内存空间之和。

3. 访问方式：
   * Union：可以通过任何一个成员来访问共享的内存空间。
   * Struct：需要通过成员名来访问各自的内存空间。

下面是一个简单的示例来说明这些区别：

```cpp
#include <iostream>

// 使用union
union MyUnion {
    int x;
    double y;
};

// 使用struct
struct MyStruct {
    int x;
    double y;
};

int main() {
    MyUnion u;
    MyStruct s;

    u.x = 10;
    std::cout << "Union value: " << u.y << std::endl;  // 输出的值也是10，因为x和y共享内存

    s.x = 10;
    s.y = 20;
    std::cout << "Struct values: " << s.x << " " << s.y << std::endl;  // 输出的值分别是10和20

    return 0;
}
```

在上面的示例中，我们创建了一个union和一个struct，然后分别对它们的成员进行赋值和访问。可以看到，union的成员x和y共享内存，而struct的成员x和y各自有独立的内存空间。
